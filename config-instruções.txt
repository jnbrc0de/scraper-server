# Instruções para configurar o proxy BrightData

1. Primeiro, verifique se o arquivo "proxies.json" foi criado corretamente na raiz do projeto. Ele deve conter:

```json
[
  {
    "id": "brightdata_proxy",
    "url": "http://brd-customer-hl_aa4b1775-zone-bytebuy:j87j7wu0492w@brd.superproxy.io:33335",
    "type": "http",
    "country": "US",
    "city": "New York",
    "isp": "BrightData",
    "tags": ["shopping", "residential"],
    "enabled": true
  }
]
```

2. Edite manualmente o arquivo .env na raiz do projeto e certifique-se de que as configurações de proxy estão como abaixo:

```
# Configurações de Proxy
USE_PROXIES=true
PROXY_FILE=./proxies.json
PROXY_ROTATION_STRATEGY=sequential
PROXY_HEALTH_CHECK_INTERVAL=900000
PROXY_HEALTH_CHECK_URL=https://httpbin.org/ip
PROXY_BAN_DURATION=1800000
```

3. Reinicie o servidor após fazer essas alterações.

4. Para testar se o proxy está funcionando, você pode usar o seguinte comando:

```
node -e "const axios = require('axios'); const proxyUrl = 'http://brd-customer-hl_aa4b1775-zone-bytebuy:j87j7wu0492w@brd.superproxy.io:33335'; const httpsAgent = new (require('https-proxy-agent'))(proxyUrl); axios.get('https://httpbin.org/ip', { httpsAgent }).then(res => console.log(res.data)).catch(err => console.error(err))"
```

IMPORTANTE: O proxy BrightData geralmente tem limites de uso, então monitore o consumo para evitar cobranças inesperadas.
